Sigthe é um sistema de controle de sessões de hemodialise, que visa abranger as necessidade de cadastro e controle de todo o precesso, desde o cadastramento dos profissionais envoldidos e pacientes, ao registro de sessões de hemodialise e consultas médicas.

Apresentação do trabalho: [SigThe](https://github.com/Ellen-Tuane/SigThe/blob/master/Apresenta%C3%A7%C3%A3o/Apresenta%C3%A7%C3%A3o_sigthe.pdf)

O sistema pode ser compilado utilizando a IDE netbeans.

Product Version: Apache NetBeans IDE 13
Java: 11.0.15; OpenJDK 64-Bit Server VM 11.0.15+10-Ubuntu-0ubuntu0.20.04.1

Ao carregar o projeto na IDE, a janela que primeiro deve-se inicializada é chamada de J_Main.java.

Apartir da inicialização desta janela, as demais serão automaticamente inicializada conforme necessidade.

O arquivo modelo_fisico_banco_dados.mwb pode ser inicializado utilizando o mysql workbanch. Após a inicialização do arquivo, o banco de dados pode ser automaticamente criado seguindo os passo descritos no link: https://dev.mysql.com/doc/workbench/en/wb-forward-engineering-live-server.html.  Caso seja necessário, as query para contrução do banco de dados pode ser consutadas no arquivo Código de criação das tabelas.sql.
